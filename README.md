# 🍷 ボトルキープ管理

顧客のボトルキープを効率的に管理するPWAアプリです。

## 特徴

- 📱 **スマホ対応** — ホーム画面に追加してアプリのように使える
- 🔒 **プライバシー重視** — データはすべて端末にローカル保存。サーバー通信なし
- 📴 **オフライン対応** — ネット接続なしで動作
- 💾 **バックアップ** — JSONファイルでエクスポート/インポート（iCloud, Google Drive等に保管）

## 機能

| 機能 | 説明 |
|------|------|
| 👤 顧客管理 | 名前・電話番号・メールアドレス・メモの記録 |
| 🍾 ボトル管理 | 銘柄・カテゴリ・残量・預かり日・保管期限の管理 |
| 📍 保管場所管理 | 棚・エリアごとの管理、収容状況の可視化 |
| 📊 ダッシュボード | 期限間近のボトル警告、最近の登録一覧 |
| 🔍 検索・フィルター | カテゴリ・ステータスで絞り込み |

## 使い方

1. [GitHub Pages のURL] をスマホのブラウザで開く
2. 「ホーム画面に追加」でアプリとして利用
3. まず **保管場所** → **顧客** → **ボトル** の順に登録

## データについて

- すべてのデータは端末の `localStorage` に保存されます
- 開発者がユーザーのデータにアクセスする手段はありません
- バックアップは設定画面からJSONファイルとしてダウンロードできます
- ご自身のクラウドストレージ（iCloud, Google Drive等）に保管してください

## 技術構成

- HTML / CSS / JavaScript（フレームワーク不使用）
- PWA（Service Worker + Web App Manifest）
- データ保存: localStorage
