# 🍷 ボトルキープ管理

バー・レストランなどで顧客のボトルキープを管理するPWAアプリです。

🔗 **アプリURL**: [https://aboshidaisuke.github.io/bottle-keep/](https://aboshidaisuke.github.io/bottle-keep/)

---

## 📱 スマホでの使い方

1. 上記URLをスマホのブラウザで開く
2. 「ホーム画面に追加」でアプリとしてインストール
   - **iPhone**: Safari → 共有ボタン（□↑）→「ホーム画面に追加」
   - **Android**: Chrome → メニュー（⋮）→「ホーム画面に追加」
3. ホーム画面のアイコンからアプリとして起動

---

## ✨ 特徴

| 特徴 | 説明 |
|------|------|
| 📱 **アプリのように動作** | ホーム画面に追加すればアドレスバーなしの全画面表示 |
| 📴 **オフライン対応** | ネット接続なしで動作。お店のWi-Fiが不安定でも安心 |
| 🔒 **プライバシー重視** | データはすべて端末にローカル保存。サーバー通信なし |
| 💾 **バックアップ対応** | JSONファイルでエクスポート → 各自のクラウドに保管 |
| 📲 **iPhone / Android 両対応** | どちらのスマホでも同じように使える |

## 🎯 主な機能

### 👤 顧客管理
- 顧客の登録・編集・削除
- 名前・電話番号・メールアドレス・メモを記録
- 検索・ソート（名前順 / ボトル数順 / 登録日順）

### 🍾 ボトル管理
- ボトルの登録・編集・削除
- 銘柄名・カテゴリ（ウイスキー / ワイン / 焼酎 / 日本酒 / ブランデー / その他）
- 顧客と保管場所への紐付け
- 預かり日・保管期限・残量（%）の管理
- ステータス管理（保管中 / 空き / 期限切れ）
- カテゴリ・ステータスでフィルター

### 📍 保管場所管理
- 場所の登録・編集・削除
- 場所名・エリア・最大収容本数を記録
- 使用率バーで空き状況を可視化

### 📊 ダッシュボード
- 全体の統計（顧客数・ボトル数・場所数・期限間近数）
- 期限間近のボトル警告リスト
- 最近の登録ボトル一覧

### ＋ インライン登録
- ボトル登録画面から、顧客や場所がまだない場合その場で追加可能
- 追加後は自動でボトル登録画面に戻り、追加した項目が選択される

### ⚙️ 設定
- バックアップ（JSONダウンロード）
- 復元（JSONファイルから読み込み）
- 全データ削除

## 🔐 データとプライバシー

- すべてのデータは端末の `localStorage` に保存されます
- **サーバーへのデータ送信は一切ありません**
- 開発者がユーザーのデータにアクセスする手段はありません
- バックアップは設定画面からJSONファイルとしてダウンロードできます
- ご自身のクラウドストレージ（iCloud Drive / Google Drive等）に保管してください
